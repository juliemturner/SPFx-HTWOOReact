# spfx-htwooreact

## Summary

This is a very basic sample of setting of an SPFx project to use HTWOO-React.

[HTWOO-React](https://lab.n8d.studio/htwoo/htwoo-react/?path=/story/introduction-getting-started--page)

## Minimal Path to Awesome

- Clone this repository
- Ensure that you are at the solution folder
- in the command-line run:
  - **npm install**
  - **gulp serve**
